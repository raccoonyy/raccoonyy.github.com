[RFDP] django-flat-theme

raccoony | 10 Apr 2015

RFDP : Raccoony's Favorite Django Plugins

지난 번에는 Django 관리자 화면을 조금 예쁘게 바꿔주는 django-wpadmin을 소개해드렸었는데요. 이것보다 설정이 더 단순하고 모양도 꽤 괜찮은 플러그인이 나와서 소개해봅니다.

플러그인 이름 : django-flat-theme

저장소 : https://github.com/elky/django-flat-theme

모습

설치

pip install django-flat-theme

설정

settings.py의 INSTALLED_APPS에서 django.contrib.admin보다 앞에 flat을 추가합니다.

INSTALLED_APPS = (
    ...
    'flat',
    'django.contrib.admin',
    ...
)

주의점

Django 1.6 이상에서 작동하고, 그 이전 버전에서는 아이콘이 좀 달라 보일 수 있다고 하네요.
