프로젝트

도도 테이블 by 스포카

SQLAlchemy에서 가져온 결과를 HTML <table> 태그로 쉽게 보여주는 파이썬 라이브러리를 스포카에서 공개했습니다. 만든 과정을 담은 블로그 글에도 유익한 정보가 많네요.

블로그

결제 모듈 I'mport 연동하기 1편 - 개요 by 권재원 님

간편한 결제 API를 제공하는 I'mport를 Django에서 사용하는 방법을 권재원 님이 정리하였습니다.

  • 2편 - I'mport와 연동할 모듈 작성
  • 3편 - 뷰 작성
  • 4편 - 템플릿 작성

병렬처리 by 불곰님

파이썬의 GIL(Global Internal Lock) 문제를 해결하고자 파이썬 3.2부터 추가된 concurrent.futures 모듈을 불곰님이 예시와 함께 설명합니다.

코드로 읽는 NumPy 튜토리얼 by shia님

행렬이나 삼각함수 등 다양한 계산을 쉽게 해주는 NumPy 라이브러리에 대한 간단한 튜토리얼을 shia님이 정리하였습니다.

Django에서 외부 DB에도 ORM 적용하기 by 권재원 님

inspectdb 명령을 통해, 처음부터 Django로 만들지 않은 DB에 대해서도 ORM 기능을 사용하는 방법을 권재원 님이 알려줍니다.

강좌

(온라인) 데이터 과학을 위한 Python 입문

파이썬 입문자를 위한 온라인 공개 강좌를 가천대학교의 최성철 교수님이 K-MOOC 사이트에 공개하였습니다. 3월 20일부터 시작합니다. (수강신청 마감: 3월 31일)

강의 영상은 유튜브에도 공개되어 있습니다.

행사

파이콘 한국 2017 키노트 스피커 추천

파이콘 한국 2017 행사에서 키노트를 맡을 연사를 추천받습니다. 국적이나 인종, 나이, 성별에 구애 없이, 파이썬 생태계에 많은 발전과 공헌을 해주신 분을 추천해달라고 합니다. (연사 추천 외에 파이콘 한국 준비위원회에 요청하고 싶은 내용을 적을 수도 있습니다.)
