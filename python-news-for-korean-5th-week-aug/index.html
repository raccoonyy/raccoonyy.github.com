한국의 파이썬 소식(8월 다섯째 주)

한국에서 일어나는 파이썬 관련 소식을 전합니다.

패키지

aWSGI - Asynchronous WSGI server by 박영록 님

uvloop와 httptools로 만든 비동기형 wsgi 서버입니다. 아래는 박영록 님이 올리신 프로젝트 소개 글입니다.

블로그

Python에서 _(밑줄)의 의미 by 이정우 님

__init__.py 파일이나 _method 같은 메서드에는 왜 _(밑줄)이 사용되고, 그것이 어떤 의미인지 이정우 님이 정리했습니다.

Jupyter notebook markdown에 내가 원하는 css 적용하기 by 윤석준 님

주피터 노트북에 마크다운으로 작성한 부분에 css를 입히려는 시도와 삽질기를 윤석준 님이 알려줍니다.

Python에서 Segmentation Fault 발생시 gdb로 stack trace 해보기 by 지훈현서 님

파이썬 프로그램이 죽으면서 아무런 정보를 주지 않을 때가 있는데요. gdb를 사용하여 어느 부분이 문제가 되는지 찾을 수 있는 방법을 지훈현서 님이 알려줍니다.

에버노트로 블로깅하기 by 이진석 님

에버노트에 글을 쓰고, 다 쓴 글에 태그를 붙이면 알아서 블로그에 퍼블리싱되는 시스템을 이진석 님이 구축해보았습니다.
