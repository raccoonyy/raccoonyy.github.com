파이썬, Django와 관련된 국내 블로그

raccoony | 07 Apr 2015

지난 번 유용한 리소스 글에 이어, 이번에는 블로그를 소개합니다.

ChangeLog
---------
2016-01-30 devissue님의 블로그 추가

파이썬이나 Django가 모두 외국 개발자들의 산물이다보니 대다수 자료가 영문으로 작성되어 있는 현실에, 한글로 고급 기술을 정리해주는 분들의 블로그는 정말 가뭄에 샘물 같습니다.

이런 귀한 블로그들을 찾아보았습니다(가나다 순).

  • 박영록 님의 블로그 - http://www.codeok.net

소프트웨어 아웃소싱 그룹인 이콜레모의 멤버인 박영록 님의 블로그입니다. 박영록 님의 개인 위키에도 생각해볼 만한 글이 많습니다.

추천 글 : [스타트업을 위한 기술 스택](http://www.codeok.net/%EC%8A%A4%ED%83%80%ED%8A%B8%EC%97%85%EC%9D%84%20%EC%9C%84%ED%95%9C%20%EA%B8%B0%EC%88%A0%20%EC%8A%A4%ED%83%9D)

스타트업이라면 파이썬과 Django를 선택해야 하는 이유를 설명합니다. 비슷한 기술인 루비 온 레일스와 살짝 비교도 되고 있습니다.

  • 스포카 블로그 - https://spoqa.github.io/

도도포인트라는 서비스를 운영하는 회사인 스포카의 기술 블로그입니다. 실제 서비스에서 파이썬을 어떻게 활용하고 있는지 알 수 있습니다.

추천 글 : Jinja2 API를 이용한 템플릿 분석

  • 파이님

블로그 1 - https://perhapsspy.wordpress.com/

블로그 2 - http://perhapsspy.tumblr.com/ 

슬라이드 - http://www.slideshare.net/perhapsspy/

스마트스터디의 서버 개발자인 파이님의 블로그+슬라이드입니다. 어느 시점엔가 블로그가 둘로 나뉘었는데요. Django와 관련된 삽질(?)기는 워드프레스 쪽에 많습니다.

추천 슬라이드 : [Django, 저는 이렇게 씁니다](http://www.slideshare.net/perhapsspy/django-42665652)

Django로 서비스를 운영할 때 겪을 만한 고민들에 대해 적절할 수도 있는(환경마다 조금 다를 수도 있지요) 대답을 주는 슬라이드입니다.
  • 한날님 블로그 - http://blog.hannal.com/

날로 먹는 Django 프레임워크 강좌라는 Django 강좌를 진행하고 계신 한날님의 블로그입니다. 해당 강좌는 인스타그램과 비슷한 서비스를 만들면서 Django의 기능들을 익히는 형태로 구성되어 있습니다. 2015년 4월 현재 5화까지 공개되었네요.

추천 글 : 날로 먹는 Django 프레임워크 강좌 여는 글

이 외에도 개발 생활이라는 연재 글을 통해 서버 개발자의 삶을 엿볼 수 있습니다.

  • 홍민희 님 블로그 - http://blog.dahlia.kr/

파이썬 이미지 라이브러리인 Wand로 유명하신 홍민희 님의 블로그입니다. 파이썬 뿐만 아니라 생활 속 깊은 고민들이 묻어납니다.

추천 글 : 디자이너에게 부탁하기

파이썬 블로그라고 소개했지만 추천 글은 '디자이너와 소통 문제'를 다룬 글로 선정했습니다. 개발자가 간과하기 쉬운 문제를 적절히 다루고 있습니다.

  • devissue님 블로그 - https://devissue.wordpress.com

Django RESTful Framework를 잘 활용하시는 devissue님의 블로그입니다. PyCharm의 설치/설정부터 RESTful API를 구축해보고, 이후 게시판까지 만드는 과정을 자세하게 설명해줍니다.

추천 글 : PyCharm 설치부터 REST API 구축, 게시판 만들기까지(시리즈)

  • Jongrae님 블로그 - http://myjr52.tumblr.com/

[공학자를 위한 파이썬]이라는 연재물을 올리고 계신 Jongrae님의 블로그입니다. [공학자를 위한 파이썬]은 컴퓨터 관련 전공이 아닌 공학도들이 수치해석이나 그래프, 통계에 파이썬을 사용할 수 있게 알려주는 연재물입니다.
