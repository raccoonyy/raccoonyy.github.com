유용한 파이썬, Django 리소스

raccoony | 03 Apr 2015

파이썬과 Django로 개발할 때는 구글과 스택오버플로를 항상 끼고 살게 마련인데요. 이 외에도 개인적으로 유용했던 리소스들을 소개해 봅니다.

웹사이트

  • 구글 - https://www.google.com/

    파이썬 패키지를 소개한 pypi 사이트가 존재하긴 하지만 검색하기엔 오히려 구글이 좋습니다. 원하는 기능에 python만 덧붙여 검색하면 못 찾을 게 없을 정도지요. (ex. 트위터 모듈이 필요하다면 python twitter로 검색합니다.)

  • Django 공식 홈페이지 - https://www.djangoproject.com/

    얼마 전 디자인을 바꾸면서 세련미까지 더한 공식 홈페이지입니다. Django 공식 문서만 읽으면 Django를 다 익혔다고 할 정도입니다. (영어라서 그게 문제...)

  • PyCon Korea 2014 강연 목록 - http://www.pycon.kr/2014/programs/list/

    2014년에 열렸던 파이콘 코리아의 강연 목록입니다. 발표 자료와 동영상이 공개되어 있어서 얼마나 고마운지 모르겠습니다.

  • Django 패키지 목록 - https://www.djangopackages.com/

    Django용 패키지를 검색할 수 있는 서비스입니다. 키워드로 검색을 하면 관련된 패키지들의 사용 현황, fork나 star 개수 등도 한 눈에 파악할 수 있습니다.

튜토리얼

튜토리얼은 굉장히 많지만 여기서는 좀 내용이 특색 있는 경우만 소개합니다.

  • TDD Django 튜토리얼 확장판 - http://chimera.labs.oreilly.com/books/1234000000754/ch01.html

    저자가 예전에 썼던 간단한 튜토리얼을 확장하여 오렐리에서 책으로 냈는데요. 온라인에서는 무료로 읽을 수 있습니다. 번역서인 클린 코드를 위한 테스트 주도 개발는 비제이퍼블리싱에서 출판했네요.

  • DjangoGirls의 튜토리얼 - http://tutorial.djangogirls.org/en/index.html

    대다수의 Django 튜토리얼은 독자가 파이썬이나 웹 정도는 개념을 잡은 독자라고 가정하는데요. 이 튜토리얼은 '웹의 기본 개념'과 '셸에서 명령어 입력하는 법'부터 알려줍니다. 그야말로 왕초보를 위한 튜토리얼이라고 할 수 있겠네요.

최신 소식

  • 파이썬 위클리(메일링) - http://www.pythonweekly.com/

    파이썬과 관련된 좋은 글, 최신 모듈, 구인 정보 등을 매주 이메일로 보내줍니다. 가끔은 읽기 버겁다 싶을 정도로 많은 내용이 담겨 있지만, 그만큼 놓치기엔 아까운 유용한 정보가 가득 담겨 있습니다.

  • 파이썬 위클리(트위터) - https://twitter.com/pythonweekly

    파이썬 위클리(메일링)의 트위터 계정입니다. 위클리 메일을 발송한 다음 날부터 위클리의 내용을 하나씩 올려주더군요. 메일링 가입이 부담스러운 분께 추천합니다.
